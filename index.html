    About Smart Aid

    Smart Aid is a revolutionary tool designed to assist visually impaired individuals in reading and understanding text through image capture and multi-language translation.

    Key Features:

    • Real-time text recognition from images
    • Multi-language translation support
    • Text-to-speech functionality
    • High contrast modes for better visibility
    • Keyboard navigation support
    • Screen reader optimization

    Help & Support

    Keyboard Shortcuts:

    • Alt + S - Focus search box
    • Alt + R - Refresh data
    • Alt + T - Toggle theme
    • Alt + A - Toggle accessibility panel
